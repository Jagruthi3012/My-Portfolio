# 🌐 Portfolio Website – Jagruthi Kelavalli Jagadish

This repository contains the source code for my personal **portfolio website**, built to showcase my skills, projects, and experience as an aspiring Backend Engineer.  

## 🚀 Tech Stack
- **Frontend:** React.js, Vite, Tailwind CSS  
- **Backend/Integration:** Node.js, EmailJS (for contact form)  
- **Deployment:** GitHub Pages / Vercel  

## 📌 Features
- **Responsive Design** – Optimized for desktop and mobile.  
- **Dark/Light Mode** – Toggle support with Tailwind CSS.  
- **Hero Section** – Brief introduction with professional tagline.  
- **About Me** – Academic and professional journey.  
- **Skills** – Technical stack organized by categories.  
- **Projects Showcase** – Highlighted projects with descriptions:
  - Virtual Classroom (Socket Programming)  
  - Women & Child Safety Management System (MERN stack)  
  - Facial Description-Based Suspect Detection (ML, PyTorch, Keras)  
  - Cab Aggregator (PHP, MySQL)  
  - Concurrent TCP Chat System (C++)  
  - 16-bit Custom Processor (Verilog, FPGA)  
- **Contact Section** – Interactive form integrated with EmailJS.
